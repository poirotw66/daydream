# #003 詐欺調查 Copilot（IEEE-CIS）

> 定稿日：2026-06-25｜狀態：**候選池**

## 一句話

用 [IEEE-CIS Fraud Detection](https://www.kaggle.com/competitions/ieee-fraud-detection) **公開**交易列，做 **調查 Copilot**：工具查特徵分布、比對同類案件、產出**調查備忘錄草稿**；**不**自主判定攔截／放行。

## 轉職訊號

展示 **tabular 風控 + Agent 編排**：模型分數僅作輸入之一，Agent 負責 **解釋、檢索類案、整理調查步驟** — 呼應 [kaggle-經典賽與學習路徑](/queries/kaggle-經典賽與學習路徑.md) P3。（推測）

## 7 天 MVP

| 日 | 交付 |
|----|------|
| Day 1 | 下載 IEEE-CIS 子集；訓練簡單 baseline（LightGBM）或沿用公開 notebook 分數 |
| Day 2–3 | Tools：`explain_row`、`similar_cases`、`feature_summary` |
| Day 4 | Agent 流程：輸入 `transaction_id` → 備忘錄 markdown |
| Day 5 | 3 個固定案例 golden（人工撰寫調查要點 checklist） |
| Day 6–7 | README 強調 **human-in-the-loop** + 不做 production block |
