# 每日資訊收集 Agent

## Summary

擁有者計畫創建**個人每日 agent**，自動（或半自動）**收集所關注的資訊來源**，餵給 [每日技術前沿-blog](/concepts/每日技術前沿-blog.md) 與 [金融業-ai-agent-side-發想](/concepts/金融業-ai-agent-side-發想.md)。（確定）[2026-06-25-學習與-side-主軸調整](/sources/2026-06-25-學習與-side-主軸調整.md)

## Key Points

- 定位：side project／工具鏈一環，非正職系統。（推測）[2026-06-25-學習與-side-主軸調整](/sources/2026-06-25-學習與-side-主軸調整.md)
- 開發可採 [cursor-superpowers](/entities/cursor-superpowers.md) 等 Agent skill 模式。（確定）[2026-06-25-學習與-side-主軸調整](/sources/2026-06-25-學習與-side-主軸調整.md)
- 輸出物（推測）：每日 digest、候選論文／產品連結、可寫進 bloss0m 的素材卡。
