# 現職場內 Agent 案子

## Summary

自 **2026-06-25** 起，擁有者計畫在 [國泰金控-ai-架構師](/entities/國泰金控-ai-架構師.md) 場景內**多做一個 Agent 相關案子**，作為 [年底轉職-agent-準備](/concepts/年底轉職-agent-準備.md) 的**正職主軸**之一；對外仍以 [bloss0m-com](/entities/bloss0m-com.md) 呈現可開源、可驗證作品，**不得**將未公開內部資料寫入 side project。（確定）[2026-06-25-主線重定義-agent職涯](/sources/2026-06-25-主線重定義-agent職涯.md)

## Key Points

### 目的

- 累積 **企業場景 Agent 開發／應用** 的實戰深度（編排、評估、上線、合規）。（確定）[2026-06-25-主線重定義-agent職涯](/sources/2026-06-25-主線重定義-agent職涯.md)
- 與公開 bloss0m 互補：正職 = 金融級約束與規模；公開站 = repo／文章可點驗。（推測）[2026-06-25-主線重定義-agent職涯](/sources/2026-06-25-主線重定義-agent職涯.md)

### 邊界

- 可對外描述：角色、方法論、技術棧類型、**已公開或可自行抽象**的設計模式。（推測）[2026-06-03-年底轉職與時間配置](/sources/2026-06-03-年底轉職與時間配置.md)
- 不可：未授權的內部數據、架構圖、客戶／監管敏感細節。（確定）[2026-06-03-年底轉職與時間配置](/sources/2026-06-03-年底轉職與時間配置.md)

### 與其他主線

- **AI CUP 2026** 已告一段落；競賽產出若有 harness／eval 想法，可轉為 bloss0m 或場內案子的**方法參考**，不再佔用 side 賽季時間。（確定）[2026-06-25-主線重定義-agent職涯](/sources/2026-06-25-主線重定義-agent職涯.md)
- [內容農場變現軌](/concepts/內容農場變現軌.md) **暫時收尾**。（確定）[2026-06-25-學習與-side-主軸調整](/sources/2026-06-25-學習與-side-主軸調整.md)
