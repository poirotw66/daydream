# 內容農場變現軌

## Summary

在 [雙軌個人架構](/concepts/雙軌個人架構.md) 中，**變現／產品軌** 的一條具體落地：以 **SEO 內容站累積流量與廣告變現**（[recipe-bloss0m-com](/entities/recipe-bloss0m-com.md)），與 **貼圖產品與自動化上架**（[貼圖商務自動化](/concepts/貼圖商務自動化.md) → [line-貼圖平台](/entities/line-貼圖平台.md)）並行，共用「把內容做成可規模化資產」的思維，但產品形態不同。（確定）[2026-06-03-recipe-bloss0m-內容農場](/sources/2026-06-03-recipe-bloss0m-內容農場.md)

## Key Points

| 維度 | 內容農場（食譜站） | 貼圖商務 |
|------|-------------------|----------|
| 終點產品 | 網頁 + 搜尋流量 | LINE 貼圖 |
| 變現 | AdSense（規劃中 Ads 測 landing） | 平台銷售 + 社群導流 |
| 核心能力 | 叢集 SEO、structured data、工具頁留存 | AI 生產線、上架自動化 |
| 代表實體 | [recipe-bloss0m-com](/entities/recipe-bloss0m-com.md) | [line-貼圖平台](/entities/line-貼圖平台.md) |

- 與 **知識合成軌**（[研究與知識合成](/concepts/研究與知識合成.md)、[notebooklm](/entities/notebooklm.md)）分開管理時間，避免與轉職用 Agent 作品集混為一談。（推測）[2026-06-03-recipe-bloss0m-內容農場](/sources/2026-06-03-recipe-bloss0m-內容農場.md)
- 對 **AI agent 轉職** 的建議敘事：食譜站可呈現 **內容管線 + 可量測指標**（索引、工具轉換），而非僅 SEO 勞力。（推測）[2026-06-03-recipe-bloss0m-內容農場](/sources/2026-06-03-recipe-bloss0m-內容農場.md)
